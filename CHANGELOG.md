## [1.0.0] - 2025-12-23

### Added
- First-person camera shake while freezing, with configurable rotation/translation intensities.
- Optional item-in-hand tremble while freezing.
- Configurable frost vignette overlay that intensifies with freeze progress (range/speed/disturbance controls).
- Optional whiteout post-processing effect while freezing.
- Ice crack sound effect triggered as freezing begins, with configurable volume.
- In-game configuration screen via Mod Menu (optional).
